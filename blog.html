Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM. Selecting by ID, Selecting by Class, Selecting by TagName , Selecting by QueraySelector etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText and textContent is: innerHTML can set HTML content, innerText can set text which is visible in an element and textContent can set all text including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is process that added a parent element instead of each individual child element.

Question-4: What is event bubbling in the DOM?

The event starts from the target element and moves upward to the parent element.
